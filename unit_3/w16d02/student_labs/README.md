# Random Taco!

Use fetch to the below API 

```js
   fetch('http://taco-randomizer.herokuapp.com/random/?full-tack=true')
```


to render the view of a random taco. No styling needed!

![screen shot](https://i.imgur.com/1HbPnjB.png)
